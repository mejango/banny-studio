# Phase 1: BannyCore Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Platform-free Swift package with the v2 document model, legacy v1 importer, and a deterministic scene simulator verified against the original webapp's math on real ep1 data.

**Architecture:** Pure Swift (Foundation only). Simulation runs in normalized stage space (x as 0..1 fraction of stage width; the web's `speed*(W/900)*dt` px integration ≡ `speed/900*dt` in fraction space). All state is a pure function of (events, t) via fixed 1/60 s steps. Golden fixtures generated by executing the extracted original JS math (node) on ep1's real event streams.

**Tech Stack:** Swift 6.1, SwiftPM, swift-testing (XCTest fallback), node (fixture generation only, not a runtime dep).

## Global Constraints

- Min targets: macOS 14, iOS 17 (package platforms).
- Determinism: fixed integration step `DT = 1/60`; partial final step `h = min(DT, t - tt)`; epsilon `1e-9` loop bound; turn delay 0.1 s; depth clamp [−12, 1]; edge margin fraction 0.044; depth rate `(speed/320)*0.36/max(gScale, 0.1)`; walk rate `speed/900` fraction/s; gait phase rate `speed/22` rad/s while moving; jump duration `round(460/gravity)` ms, height `30/gravity`; tilt ±9°.
- Event codes (verbatim from web): `ArrowLeft ArrowRight ArrowUp ArrowDown Comma Slash Period KeyM KeyT KeyB KeyJ` + `outfit`. Groups: move, depth, tilt, talk, blink, jump. Blink map: Comma→closed, Slash→brow1, Period→brow2.
- Defaults: speed 320, wobble 7, gScale 0.6, gravity 1, gSize 1, size presets 1/0.62/0.38, max 10 characters.
- v1 quirks to honor: `x`/`recStart.x` are fractions but legacy values >1.5 are px (divide by 900, the web's fallback width); events must be sorted by t; `speed`/`wobble` absent in v1 → defaults.
- Legacy source of truth for tests: `/Users/jango/Documents/banny/show/ep1/beat1/staging/1.json`.

### Task 2: v2 document model

**Interfaces (produced):**
```swift
public struct ShowDocument: Codable, Equatable { public var version: Int // = 2
  public var scenes: [Scene]; public var show: [ShowSegment]; public var settings: Settings }
public struct Settings: Codable, Equatable { public var activeScene: Int; public var lightSize: Double }
public struct Scene: Codable, Equatable, Identifiable { public var id: String; public var name: String; public var state: SceneState }
public struct SceneState: Codable, Equatable { public var characters: [Character]; public var audioTracks: [AudioTrack]
  public var lights: [Light]; public var cropAnchors: [Double]; public var gScale, gravity, gSize: Double
  public var background: BackgroundSpec? }
public struct Character: Codable, Equatable { public var body: Body; public var x, depth, size: Double
  public var face: Int; public var baseOutfit: [Int: String]; public var subs: [Subtitle]
  public var clips: [AudioClip]; public var events: [PerfEvent]; public var armedGroups: Set<EventGroup>
  public var name: String; public var trackFx: Fx; public var recStart: StartPose?
  public var speed: Double // = 320
  public var wobble: Double // = 7
}
public struct StartPose: Codable, Equatable { public var x, depth: Double; public var face: Int }
public enum PerfEvent: Codable, Equatable {   // custom coding: {t,code,phase} | {t,outfit:{slot,name}}
  case key(t: Double, code: EventCode, down: Bool)
  case outfit(t: Double, slot: Int, name: String?) }
public struct AudioClip: Codable, Equatable { public var id, name: String
  public var start, dur, offset, srcDur: Double; public var fx: Fx }
public struct Fx: Codable, Equatable { public var gain, low, mid, high, reverb: Double; public var pan: Pan }
public enum Pan: Codable, Equatable { case follow, narrow, wide, value(Double) }
public struct Light: Codable, Equatable { public var x, y: Double } // normalized 0..1
public enum BackgroundSpec: Codable, Equatable { case image(file: String, crop: Crop), video(file: String, crop: Crop) }
public enum Crop: String, Codable { case cover, fit, stretch, tile }
public enum Body: String, Codable, CaseIterable { case orange, original, pink, alien }
```

public struct CharacterPose: Equatable {

  public var eye: EyeExpression; public var talking: Bool

  public func pose(characterIndex: Int, at t: Double) -> CharacterPose }

Jump: scan for last KeyJ down ≤ t; active if `t < tj + round(460/gravity)/1000`; progress `(t−tj)/dur`. Eye/tilt/talk/held from resetToTime scan. Outfit: baseOutfit overlaid with outfit events ≤ t (name nil removes). Subtitle: latest-starting sub covering t.
